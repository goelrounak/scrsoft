QUICKSOL FOR STORAGE SOLUTIONS MANUFACTURING

Storage Solutions Manufacturers processes the enquiries manually and design the unit components and draws the layout using any CAD tools.

The project titled the QuickSol aims at generating BOM and Elevation faster, without error and minimizing manual intervention. QuickSol is used to workout the offer, which comprises of civil plan, rack layout, elevation, BOM (in various formats) and summary of the solution for all products like HDS, MF, Mobile, Drive In etc.

The software has facility to store the standard component details and create drawings required for design. Database will be created for standard components along with the description and price. This program will facilitate proper selection of components to arrive at the optimum design. If any modifications or changes in drawing will be simultaneously reflected in the Bill Of Materials and part description. This Proposed system will fulfil the features mentioned below:

  • No Need Of AutoCAD knowledge (i.e.: even a layman can create an effective product design).
  • Designing can be Automated and performed by the software with very less time.
  • Very Less Time is Consumed to Generate the Drawing.
  • Software is Designed in User-friendly Manner.
